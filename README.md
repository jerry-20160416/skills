# Agent Skills Collection

这是一个 Agent Skills 仓库，包含各种有用的技能包。

## Skills 列表

### alibaba-java-manual

**阿里巴巴 Java 开发手册（黄山版）**

包含完整的 Java 开发规范，涵盖：

- 编程规约（命名风格、常量定义、代码格式、OOP规约、日期时间、集合处理、并发处理、控制语句、注释规约、前后端规约）
- 异常日志（错误码、异常处理、日志规约）
- 单元测试
- 安全规约
- MySQL 数据库（建表规约、索引规约、SQL语句、ORM映射）
- 工程结构（应用分层、二方库依赖、服务器）
- 设计规约

**安装方式：**

```bash
npx skills add jerry-20160416/skills@alibaba-java-manual
```
