## 5. 接口需求

### 5.1 接口列表

| 接口名称 | 请求方式 | 接口路径 | 说明 |
|----------|----------|----------|------|
| 查询列表 | GET | /api/v1/[module] | 分页查询列表 |
| 查询详情 | GET | /api/v1/[module]/{id} | 根据ID查询详情 |
| 新增数据 | POST | /api/v1/[module] | 新增数据 |
| 更新数据 | PUT | /api/v1/[module]/{id} | 更新数据 |
| 删除数据 | DELETE | /api/v1/[module]/{id} | 删除数据 |

### 5.2 接口详细定义

#### 接口1：查询列表

**请求参数**：

```json
{
  "pageNum": 1,
  "pageSize": 10,
  "keyword": "搜索关键词",
  "status": 1
}
```

**响应结果**：

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "total": 100,
    "list": [
      {
        "id": 1,
        "name": "示例数据",
        "status": 1,
        "createTime": "2026-04-08 21:00:00"
      }
    ]
  }
}
```

## 6. 约束条件

### 6.1 技术栈

- **后端框架**：Spring Boot 2.x
- **数据库**：MySQL 8.0
- **缓存**：Redis 6.x
- **消息队列**：RabbitMQ

### 6.2 编码规范

- 遵循《阿里巴巴Java开发手册》
- 使用统一的代码格式化配置
- 单元测试覆盖率 >= 80%

### 6.3 项目结构

```
project/
├── src/main/java/
│   ├── controller/    # 控制器
│   ├── service/       # 业务层
│   ├── repository/    # 数据层
│   └── entity/        # 实体类
├── src/main/resources/
│   ├── application.yml
│   └── mapper/        # MyBatis映射文件
└── src/test/java/     # 测试代码
```
